# Runbook

## Responsibilities
- provision Drive folder structure for teams, franchises, and workflows
- create Docs and Sheets from approved templates
- send governed Gmail drafts or notifications
- create Calendar reminders and follow-up events
- mirror approved runtime outputs into Workspace artifacts

## Guardrails
- no direct writes to canonical lead truth
- no direct writes to CRM systems without runtime approval
- no auth ownership
- all side effects must be attributable to a runtime event, approved admin action, or scheduled workflow

## Priority integrations
1. Gmail draft generation for lead follow-up
2. Calendar reminder creation for next-best action
3. Drive folder scaffolding for reps, managers, owners, and franchises
4. Docs generation for lead summaries, SOPs, and briefings
5. Sheets mirrors for scorecards and benchmark exports
