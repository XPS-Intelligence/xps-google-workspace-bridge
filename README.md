# xps-google-workspace-bridge

## Purpose
Google Workspace coordination layer for Drive, Gmail, Calendar, Docs, Sheets, and Apps Script bridge logic across the XPS Intelligence ecosystem.

## Best-fit role
This repository is the collaboration and document-action sidecar for:
- `xps-intelligence-system`
- `xps-intel`
- `xps-distallation-system`
- `xps-vision`

## Source of truth
- GitHub defines code, workflows, contracts, and automation law
- Railway hosts runtime systems
- Postgres and Redis remain the canonical operational data plane
- Google Workspace is the document, messaging, scheduling, and collaboration plane
